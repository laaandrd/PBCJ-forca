# PBCJ-forca

Jogo desenvolvido para a matéria de **Programação Baseada em Componentes para Jogos** da [UFABC](https://www.ufabc.edu.br/).

## O jogo
O Jogo é um jogo da forca onde o jogador deve tentar acertar a palavra antes que suas chances acabem. A cada letra errada as chances diminuem. 

A primeira pagina é o **Menu Principal** onde é possivel iniciar o jogo ou ver os créditos.

Ao iniciar o jogo uma lista de **?** apareceram indicando as letras a serem descobertas. No canto inferior há uma lista de letras já usadas. A cada letra digitada ela é adicionada a essa lista. Caso a letra digitada seja correta a posição **?** correspondente irá mudar para a letra correta. Caso contrário o número de chances diminuirá

Caso acerte a palavra uma página dizendo Salvo irá aparecer. Caso a quantidade de chances se esgotem uma dizendo Enforcado irá aparecer. Em ambas as páginas há a opção de **Jogar Novamente** ou **Voltar ao Menu Principal**.

Na página de Creditos há os criadores do jogo com os seus devidos RAs. Além de os links da onde foram tirados as músicas e sons tocados no jogo.
